# IT 运维管理平台

基于 **Go (Gin) + Vue 2 (Element UI)** 的企业级 IT 运维管理平台，涵盖资产管理、安全合规、权限治理、日程协作等核心功能，支持 LDAP 统一认证与双控审批机制。

---

## 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | Vue 2.7 + Element UI + Vue Router + Axios + ECharts + anime.js |
| **后端** | Go 1.25 + Gin + GORM + JWT + LDAP + RSA/AES 加密 |
| **数据库** | MySQL |
| **部署** | HTTPS (TLS) + 交叉编译 Linux 二进制 |

## 功能模块

### 数据看板
- 汇总统计面板，展示资产、安全、权限等关键指标

### 基础设施管理
- **区域管理** — 管理数据中心 / 办公区域
- **操作系统管理** — 维护操作系统类型字典
- **资产管理** — IT 资产全生命周期管理（CRUD + 关联区域 / OS 类型）

### 安全合规
- **漏洞扫描** — 扫描记录管理、整改报告上传、整改闭环追踪
- **渗透测试** — 渗透测试报告上传与管理
- **防火墙检查** — 检查记录与整改报告管理
- **安全整改记录** — 统一的安全整改台账
- **系统加固** — 加固检查表导出与历史记录
- **补丁更新** — 补丁记录与合规性报表
- **病毒控制** — 病毒防护记录管理
- **备份管理** — 备份申请、恢复还原记录、模板管理

### 变更管理
- 变更记录创建 / 审批 / 归档
- 变更类型自定义与排序
- 变更模板管理（上传 / 下载 / 预览）
- 变更记录扫描件管理

### 身份与权限治理
- **LDAP 统一认证** — 对接企业 AD 域控，单点登录
- **双 Token 无感刷新** — Access Token + Refresh Token 机制
- **双控审批** — 关键写操作需第二人审批验证
- **岗位权限设置** — 基于岗位的权限规则配置
- **用户权限一览** — 全量用户权限查看与导出
- **月度检查历史** — 权限月度确认记录
- **季度检查历史** — 第三方应用季度检查
- **用户变更记录** — 用户权限变更审计追踪

### SFTP 管理
- **SFTP 服务器管理** — 服务器信息维护
- **SFTP 账号管理** — 账号 CRUD + 确认表导出

### 软件与补丁
- **核准软件目录** — 企业核准软件清单管理
- **资产对应表** — 资产与软件安装映射
- **补丁更新记录** — 补丁合规性追踪

### 协作与知识
- **日程管理** — 多视图日历（月 / 周 / 日）、重复日程、日程冲突检测、通知系统
- **IT 指南** — 步骤式操作指南，支持图文 / 视频，公开访问 + 管理端编辑
- **表单发布** — 表单上传 / 发布 / 公开下载，支持跨模块引用
- **密码本** — AES 加密存储，分类管理，收藏排序，查看日志审计

### 审计日志
- **登录日志** — 用户登录记录
- **操作日志** — 关键操作审计追踪，支持详情查看

### IT 政策与拓扑
- **IT 政策** — 政策文档上传 / 预览 / 下载
- **网络拓扑图** — 拓扑图上传 / 预览 / 下载

## 安全特性

- **JWT 双 Token 认证** — Access Token (短期) + Refresh Token (长期)，无感刷新
- **RSA 加密传输** — 登录密码 RSA 公钥加密，后端私钥解密
- **双控审批机制** — 关键写操作需第二人验证，防止单人误操作
- **AES 密码本** — 敏感密码 AES 加密存储，按需解锁查看
- **HTTPS** — 全站 TLS 加密通信
- **LDAP 集成** — 企业域控统一认证
- **审计日志** — 全操作链路可追溯

## 快速开始

### 环境要求

- Go 1.25+
- Node.js + npm
- MariaDB 11.8+

### 后端启动

```bash
cd server

# 编辑配置文件
vim config.yml   # 配置数据库、LDAP、TLS 等

# 安装依赖
go mod download

# 编译运行
go build -o it-server .
./it-server
```

### 前端启动

```bash
cd client

# 安装依赖
npm install

# 开发模式
npm run serve

# 生产构建
npm run build
```

---

## 部署

后端支持交叉编译为 Linux 二进制：

```bash
# Linux amd64 编译
GOOS=linux GOARCH=amd64 go build -o it-server-linux .
```

前端构建产物位于 `client/dist/`，可部署至任意静态文件服务器或与后端合并部署。
